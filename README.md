# 📊 Análise Gráfica de Dados com Python

Projeto desenvolvido em Python para leitura e visualização de dados presentes em uma planilha Excel utilizando as bibliotecas **Pandas** e **Matplotlib**.

O objetivo do script é transformar dados tabulares em um gráfico visual simples, organizado e intuitivo, facilitando análises e interpretações de evolução ao longo do tempo.

---

# 🚀 Funcionalidades

- ✅ Leitura de arquivos `.xlsx`
- ✅ Separação automática das colunas
- ✅ Geração de gráfico de linha
- ✅ Visualização da evolução de valores ao longo do tempo
- ✅ Rotação automática dos períodos no eixo X
- ✅ Organização automática do layout do gráfico
- ✅ Exibição de grade para melhor leitura

---

# ⚙️ Como Funciona

O script realiza:

1. Leitura da planilha Excel
2. Captura das colunas:
   - `Periodo`
   - `Valor`
3. Criação de um gráfico de linha
4. Exibição visual dos dados

# 📈 Exemplo de Gráfico

O gráfico gerado exibe:

- Períodos organizados no eixo X
